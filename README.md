# Brick_Breakout
Bricks_Breakout is a Breakout clone in which the player must smash a wall of bricks by deflecting a bouncing ball with a paddle.

## Game_Play

Control Keys:
.Move paddle right =  right arrow
.Move paddle left =  left arrow
.Pause game = space
.restart game = y
